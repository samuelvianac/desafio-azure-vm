# Desafio - Criação de Máquina Virtual no Azure

## Descrição
Este repositório foi criado para documentar o processo de criação e configuração de uma máquina virtual no Microsoft Azure, como parte de um desafio prático.

## Objetivo
Aplicar os conceitos aprendidos em aula em um ambiente prático, utilizando o portal do Azure para provisionar uma máquina virtual e registrar cada etapa de forma estruturada no GitHub.

## Etapas realizadas

### 1. Acesso ao portal do Azure
Primeiro, foi realizado o acesso ao portal do Microsoft Azure para iniciar a criação dos recursos.

### 2. Criação da máquina virtual
No portal, foi selecionada a opção de criação de uma nova máquina virtual, definindo as configurações principais como:
- assinatura
- grupo de recursos
- nome da máquina virtual
- região
- imagem do sistema operacional
- tamanho da máquina
- usuário administrador
- regras de porta de entrada

### 3. Configuração de autenticação
Foi configurado o acesso administrativo à máquina virtual, definindo usuário e senha ou chave SSH, conforme a necessidade.

### 4. Configuração de rede
Foi criada ou associada uma rede virtual, incluindo regras de acesso para permitir conexão com a máquina.

### 5. Revisão e criação
Após revisar todas as configurações, a máquina virtual foi criada com sucesso no ambiente Azure.

### 6. Validação do recurso
Depois da implantação, foi realizada a validação do recurso no portal, verificando status, configurações principais e possibilidade de conexão.

## Aprendizados
Durante a execução do desafio, foi possível compreender melhor:
- como funciona a criação de máquinas virtuais no Azure
- a importância de configurar corretamente rede e autenticação
- como o Azure organiza recursos por grupos
- o papel do GitHub na documentação técnica de atividades práticas
